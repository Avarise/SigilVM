# Sigil Desktop Themes: Color Reference

This document defines the canonical color palettes used across Hyprland, Waybar, Kvantum, Alacritty, and other desktop components.
All color codes are in 8-bit hex RGB (or RGBA where applicable).  
Each key is formatted as `<theme>.<category>.<role>` for programmatic access.

---

## Matrix Theme
A luminous, electric style — neon green on black. Inspired by phosphor terminals and pulsing Auracite cores.

matrix.color.background      = #000000    # Deep black base
matrix.color.surface          = #0A0E14    # Slightly raised panels
matrix.color.accent           = #00FF66    # Electric neon green highlights
matrix.color.border           = #002200    # Subtle dark green borders
matrix.color.text.primary     = #C8FACC    # Soft green text for readability
matrix.color.text.secondary   = #88B89E    # Dimmed text and inactive labels
matrix.color.alert            = #FF3366    # Error/attention color (rarely used)
matrix.color.hover            = #003311    # Hover or pressed state background
matrix.color.opacity.active   = 1.0        # Full opacity (blur handled by compositor)
matrix.color.opacity.inactive = 0.9
